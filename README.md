## Функционал
- обработка ввода пользователя.
- Валидация вводимого выражения.
- Отображение результата на экране.
- Сохранение истории вычислений.

## Структура проекта
- Assets/Scripts/Application - основной core проекта
- Assets/Scripts/ExpressionModelData - пакет для работы с моделью данных выражений
- Assets/Scripts/Popup -  пакет для работы с popup окнами
- Asset/Scripts/Repositories - пакет для работы с сохранением данных

## Как запустить
1.  Откройте проект в Unity Editor.
2.  Откройте сцену `Assets/Scenes/SampleScene.unity`.
3.  Нажмите кнопку Play.
4.  В появившемся интерфейсе введите выражение для сложения в формате `число+число` (например, `5+3`).
5.  Результат вычисления отобразится на экране.
